Projects

Selected Projects

Payroll Management System

This desktop application manages the payroll system and attendance of employees efficiently. It tracks daily attendance and calculates salaries based on experience, designation, and company-specific rules. The system allows the admin to add or remove employees, ensuring up-to-date records, and generates detailed reports for salary and attendance. By automating payroll calculations and attendance tracking, the application minimizes manual errors, saves time, and provides a reliable platform for employee management and administration.

Tech stack
Java Swing / AWT MySQL JDBC

Implemented robust database design, different employee types, allowance and deduction modules, and printable salary slips.

Train Alert 24/7

We developed a web application using React.js called the Train Alert System, which notifies passengers, volunteers, and railway officers in real time about accidents near trains. The system integrates a chatbot to guide passengers during emergencies and uses YOLO for fire and hazard detection around the train. All alerts and notifications are automatically sent via email to ensure timely communication, helping authorities respond quickly while assisting passengers with safety instructions and guidance during accidents.

Tech stack
React Node.js Java Python YOLO BERT

Combines object detection (YOLO) with NLP models to analyze incident data and send timely alerts through a web dashboard.

Virtual Education Hub

Virtual Education Hub is a platform designed for students to connect and learn collaboratively. Students can create groups according to their learning needs or join existing ones, and the system uses a smart matching feature to pair students with similar learning preferences, such as those who prefer teaching with those who prefer listening. The platform includes chat, scheduled video calls, and a digital whiteboard for interactive learning, enabling students to collaborate efficiently, share knowledge, and engage in productive study sessions in a virtual environment.

Tech stack
HTML JavaScript MongoDB Machine Learning

Focus on matchmaking algorithms, real-time communication and an intuitive user experience for students.
